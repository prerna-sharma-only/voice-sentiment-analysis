# 🎤 Voice Sentiment Analysis AI

An intelligent web application that transforms **voice and text into emotional insights** using AI.

---

## ✨ Highlights

* 🎙️ Real-time voice input with seamless interaction
* ✍️ Instant text-based sentiment detection
* 🤖 Accurate emotion recognition (joy, anger, sadness, etc.)
* 🔐 Secure authentication system
* ⚡ Smooth, modern, and responsive UI

---

## 🧠 Powered By

**Flask • JavaScript • Hugging Face Transformers • SQLite**

---

## 🚀 Experience

Speak or type your thoughts and instantly discover the **emotion and sentiment behind them**.
